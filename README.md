# Anand Patel - Portfolio

A modern, responsive, and interactive developer portfolio built with Next.js, Tailwind CSS, and Framer Motion.

- **Modern UI/UX**: Glassmorphism, gradients, and smooth animations.
- **Responsive Design**: Optimized for mobile, tablet, and desktop.

- **Performance**: Optimized images, code splitting, and smooth scrolling.
- **Accessibility**: ARIA labels, focus management, and keyboard navigation.
- **SEO**: Metadata, sitemap, robots.txt, and JSON-LD structured data.

## üì¶ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/anandpatel/portfolio.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Copy `.env.local.example` to `.env.local` and add your Formspree ID.
   ```bash
   cp .env.local.example .env.local
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) with your browser.
